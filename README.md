# Daydream Ops

Luxury Creative Operating System — Daydream Production operasyon merkezi.

Brief, müşteri (CRM), tahsilat, teklif ve ayarlar. Veriler cihazda (localStorage) saklanır.

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Vercel veya statik hosting; `vercel.json` SPA yönlendirmesi içerir.
