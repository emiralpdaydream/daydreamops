DAYDREAM OPS — Bilgisayar açılınca (kısa liste)
================================================

TEK SEFER (ilk kurulum)
-----------------------
1. Bu klasörde dosya adı:  .env.local
   (.env.example dosyasını kopyalayıp adını .env.local yapın)

2. Yerel AI test için (anahtarı sohbete yapıştırmayın) — birini yeter:

   OPENAI_API_KEY=sk-proj-...   (OpenAI)
   veya
   GOOGLE_API_KEY=...           (Google Gemini / AI Studio)

3. Gmail / Google OAuth (client_secret_*.json dosyanız proje kökündeyse):

   npm run google:sync

   Bu komut JSON'dan GOOGLE_CLIENT_ID ve GOOGLE_CLIENT_SECRET değerlerini
   .env.local dosyasına yazar. client_secret*.json Git'e gitmez.

   Google Cloud Console → OAuth istemcisi → Authorized redirect URIs:
     http://localhost:5173/api/google/callback
     https://SIZIN-DOMAIN.vercel.app/api/google/callback

4. Kaydedin (Ctrl+S)

Canlı site (Vercel):
  npm run google:sync:prod   (veya değerleri elle kopyalayın)
  Project → Settings → Environment Variables → Redeploy


HER GÜN / BİLGİSAYAR YENİDEN AÇILINCA
-------------------------------------
1. Cursor'da bu projeyi açın (DAYDREAM OPS klasörü)

2. Terminal:

   npm run dev

3. Tarayıcı: http://localhost:5173

4. Şifre → Merkez (Bugün) veya alt menüden AI / Brief / Finans


AI ASISTAN & GMAIL
------------------
- Alt menü veya üst bar: AI Asistan
- Ayarlar → Bağlantılar Merkezi → Gmail → Bağla / Test
- .env değiştirdiyseniz: Ctrl+C → npm run dev tekrar


SORUN OLURSA
------------
- Canlıda AI: Vercel → OPENAI_API_KEY → redeploy
- Gmail "OAuth eksik": npm run google:sync veya Vercel'de GOOGLE_* değişkenleri
- JSON veya secret sohbete yapıştırmayın; sızdıysa Google Console'dan yeni secret üretin
