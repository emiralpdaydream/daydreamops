DAYDREAM OPS — Bilgisayar açılınca (kısa liste)
================================================

TEK SEFER (ilk kurulum)
-----------------------
1. Bu klasörde dosya adı:  .env.local
   (.env.example dosyasını kopyalayıp adını .env.local yapın)

2. Yerel AI test için (anahtarı sohbete yapıştırmayın) — birini yeter:

   OPENAI_API_KEY=sk-proj-...   (OpenAI)
   veya
   GOOGLE_API_KEY=...           (Google Gemini / AI Studio)

3. Kaydedin (Ctrl+S)

Canlı site (Vercel): aynı isimlerle Environment Variables → sonra Redeploy.


HER GÜN / BİLGİSAYAR YENİDEN AÇILINCA
-------------------------------------
1. Cursor'da bu projeyi açın (DAYDREAM OPS klasörü)

2. Terminal:

   npm run dev

3. Tarayıcı: http://localhost:5173

4. Şifre → Merkez (Bugün) veya alt menüden AI / Brief / Finans


DAYDREAM OPERATOR
-----------------
- Merkez ekranında üstte kart → Operatörü Aç
- Mobilde sağ alttaki Operator FAB veya alt menü "AI"
- Ayarlar → Bağlantılar Merkezi → OpenAI "Bağlantıyı Test Et"


SORUN OLURSA
------------
- .env değiştirdiyseniz: Ctrl+C → npm run dev tekrar
- Canlıda AI: Vercel → OPENAI_API_KEY → redeploy
- Google Drive/Gmail (sonraki faz): GOOGLE_CLIENT_ID + GOOGLE_CLIENT_SECRET
